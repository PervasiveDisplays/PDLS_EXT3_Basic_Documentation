Pervasive Displays Library Suite - Basic edition

The Pervasive Displays Library Suite is specifically designed for the Pervasive Displays e-paper screens and EXT3 extension board.

It provides a high-level interface to drive the e-paper screens with global, fast and partial update, supports capacitive touch and haptic feed-back.

Variants

The basic edition includes different variants:

  • PDLS_EXT3_Basic for monochrome and colour (black-white-red) screens;
  • PDLS_EXT3_Basic_Fast for monochrome screens with embedded fast update;
  • PDLS_EXT3_Basic_Touch for monochrome screens with capacitive touch.

The screens are connected to the EPD Extension Kit Gen 3 (EXT3 or EXT3-1) and Touch Expansion Board for EXT3 (EXT3-Touch).

Two introductory kits are also available:

  • The EPD Pico Kit (EPDK) includes a Raspberry Pi Pico RP2040, an EXT3-1 extension board and a 2.66" monochrome panel; and
  • The EPD Pico Kit Touch (EPDK-271-Touch) includes a Raspberry Pi Pico RP2040, an EXT3-1 and EXT3-Touch boards and a 2.71"-Touch monochrome panel with embedded fast update.

Documentation

The documentation includes two documents:

  • This Reference manual lists all the objects and functions, with details for their parameters and returned results.
  • The User guide focuses on how to use the different libraries, with commented examples.

Additionally, the Wiki provides a gradual introduction to the e-paper technology and how to use it.
