Detailed Description

Class for Pervasive Displays iTC monochrome touch screens with embedded fast update.

Screen controllers

  • LCD: proprietary, SPI
  • touch: proprietary, I2C
  • fonts: no external Flash
Note
All commands work on the frame-buffer, to be displayed on screen with flush()
